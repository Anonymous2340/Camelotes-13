What is HTML Header element?

The < header > element represents a container for introductory content or a set of navigational links.
  • One or More heading element(h1 - h6).
  • Logo or icon.
  • authorship informations.

You can have several < header > elements in one HTML document. However, < header > cannot be placed within a < footer >, < address > or another < header > element.
filename:Header.html
 < html >
  < body >
   < header >
    < h1 >Thi is the heading< /h1 >
    < p >This is the paragraph.< /p >
   < /header >
  < /body >
 < /html >
The < header > tag also supports the Global Attributes in HTML and Event Attributes in HTML.
