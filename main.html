What is HTML main element?

The < main > tag specifies the main content of a document.

The content inside the < main > element should be unique to the document. It should not contain any content that is repeated across documents such as sidebars, navigation links, copyright information, site logos, and search forms.

filename:main.html
 < html >
  < body >

   < main >
     < h1 >Lorem< /h1 >
     < p >Lorem ipsum dolor sit amet consectetur adipisicing elit. Cupiditate consectetur, praesentium similique deserunt necessitatibus vitae, corrupti dolore aut recusandae consequatur ratione provident odio nihil velit quo porro. Nostrum, sunt similique?< /p >
   < /main >

  < /body >
 < /html >
Note: There must not be more than one < main > element in a document.
